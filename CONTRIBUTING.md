# Contributing to Picodash

## Public-preview status

Picodash is currently a public preview. Pull requests are temporarily disabled
while the API, documentation, and maintenance workflow settle. Please do not
prepare or send a pull request yet; it will not be reviewed or accepted.

## Feedback

Issues are welcome for reproducible bugs and concrete feature requests. Please
search existing issues first and include the smallest reproduction you can.

## Local development

The repository uses Bun and Vite+:

```bash
bun install
bun run ready
```

The project requires Bun `>=1.3.14` and Node.js `>=22.12.0`. See the root
README for the workspace topology and local website commands.

## Future contributions

When pull requests reopen, this document will be expanded with the expected
branching, testing, review, and release workflow.
