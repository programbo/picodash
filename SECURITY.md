# Security policy

## Supported versions

Tweaker is currently a public preview and does not have a stable release line.
Security fixes, when needed, will target the latest `main` revision and the
latest published package release once releases begin.

| Version                  | Supported           |
| ------------------------ | ------------------- |
| `main`                   | Yes                 |
| Latest published release | Yes, once published |
| Older releases           | No                  |

## Reporting a vulnerability

Please do not open a public issue for a security vulnerability.

Use GitHub's private vulnerability reporting for this repository:

<https://github.com/programbo/tweaker/security/advisories/new>

Include the affected version or commit, reproduction steps, impact, and any
suggested mitigation. Please allow time for investigation before publicly
disclosing the issue.

If private reporting is unavailable, contact the repository owner through
GitHub and do not include sensitive details in a public issue.
