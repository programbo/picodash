# Code of Conduct

Tweaker is intended to be a welcoming, respectful, and constructive project.
Participants are expected to:

- communicate respectfully and assume good faith;
- focus criticism on ideas, code, and behavior rather than people;
- welcome different backgrounds and levels of experience; and
- avoid harassment, discrimination, threats, and personal attacks.

Project maintainers may remove comments, close conversations, or restrict
participation when behavior conflicts with these standards. Report concerns
privately through the repository owner's GitHub profile or GitHub's reporting
tools rather than in a public issue.
